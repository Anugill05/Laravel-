steps for creating basic controller
step 1- php artisan make:controller controllerName
-for making a controller suffix controller is required after name of controller
step 2- open created controller (in app->http->controller)
step 3- create function inside your controller
step 4- add imports of your controller in web.php
step 5- add corresponding routes to web.php
step 6- run your code with php artisan server with provided url
